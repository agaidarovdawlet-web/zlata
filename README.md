# zlata

Учебное Android-приложение для сотрудников энергосбытовой компании: лицевые счета, приборы учета, показания и платежи в офлайн-сценарии.

## Возможности

- поиск по клиентам, адресам, счетам и счетчикам;
- просмотр лицевых счетов, статусов и баланса;
- просмотр приборов учета и дат поверки;
- внесение новых показаний;
- просмотр платежей;
- работа с предзагруженной локальной базой.

## Стек

- Kotlin
- Jetpack Compose
- Material 3
- Navigation Compose
- Room
- SQLite assets

## Данные

Используемая база:

- `app/src/main/assets/energosbyt_plus.db`

Параметры Android:

- `applicationId`: `com.example.zlata`
- `minSdk`: 24
- `targetSdk`: 36
- `compileSdk`: 36.1

## Запуск

```bash
./gradlew assembleDebug
```

## Состояние проекта на 2 июля 2026

- найдено `12` Kotlin-файлов, Android-манифест и локальная asset-база;
- по содержимому это рабочий учебный офлайн-проект;
- текущая `.git`-папка повреждена или неполна: Git не распознает каталог как репозиторий;
- из-за этого репозиторий нужно инициализировать заново перед публикацией;
- автоматическую сборку в этой среде отдельно не подтверждал, потому сначала требуется восстановить git-состояние и затем прогнать Gradle.

## Замечания

- в публичном репозитории оставлена только реально используемая база `energosbyt_plus.db`;
- после реинициализации git стоит отдельно убедиться, что `.gitignore` не пропускает временные build-артефакты.
